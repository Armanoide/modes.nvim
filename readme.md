# modes.nvim

> Currently in alpha. Many things are broken.

Highlight UI elements based on current mode. Inspired by the recent addition of vim mode in Xcode-beta.

## Usage

```lua
use({
  'mvllow/modes.nvim',
  config = function()
    require('modes').setup()
  end
})
```
